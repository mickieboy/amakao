# Sitio web multipágina AMAKAO

Proyecto HTML, CSS y JavaScript responsive para una marca peruana de chocolates premium.

## Páginas incluidas

- `index.html`: Inicio
- `nuestra-marca.html`: Nuestra marca
- `origen-sostenibilidad.html`: Origen y sostenibilidad
- `contacto.html`: Contacto
- `chocolates.html`: Catálogo
- `carrito.html`: Carrito de compra
- `producto.html?id=origen70`: Descripción del producto

## Cómo abrirlo

1. Descomprime la carpeta.
2. Abre `index.html` en Google Chrome, Edge o Firefox.
3. Para publicarlo, sube toda la carpeta manteniendo la misma estructura.

## Funciones

- Diseño responsive para computadora, tablet y celular.
- Menú móvil.
- Filtros y búsqueda de productos.
- Página de producto dinámica mediante el parámetro `id`.
- Carrito funcional con `localStorage`.
- Formularios demostrativos sin conexión a un servidor.

## Productos disponibles en la URL

- `producto.html?id=origen70`
- `producto.html?id=selva80`
- `producto.html?id=aguaymanto`
- `producto.html?id=descubrimiento`

Los datos, precios, ingredientes y contactos son referenciales para fines académicos.
